# 🧘‍♂️ FocusVibe — Remote Work Noise Generator

**FocusVibe** is a minimalist menubar app for macOS that helps remote workers stay focused with customizable ambient soundscapes like rain, typing, and café background noise.

Built with **SwiftUI** and designed for simplicity and productivity.

---

## 🔊 Features

- 🎧 Loopable ambient sounds (Rain, Typing, Café, etc.)
- 🎚 Adjustable volume for each sound
- 🖥️ Runs silently in the macOS menubar
- 🚀 Auto-launches on system login

---

## 📦 Installation

1. [Download the `.dmg` file](https://github.com/khotcholava/focus-vibe-releases/releases/tag/v1.0.0) from the **Releases** section  
2. Drag the app into your **Applications** folder  
3. Launch FocusVibe and enter your personal focus zone
